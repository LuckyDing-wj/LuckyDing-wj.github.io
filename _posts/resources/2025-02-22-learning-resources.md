## 六、推荐学习顺序

1. **Java 基础**：先完成语法、集合、多线程等核心概念
2. **数据库与 JDBC**：理解 SQL 与数据持久化
3. **构建与版本控制**：掌握 Maven/Gradle 与 Git
4. **Web 框架**：学习 Spring + Spring MVC
5. **工程化提升**：单元测试、日志、监控
6. **微服务与分布式**：Spring Cloud 生态与治理
7. **项目实战与优化**：缓存、性能调优、部署

对应学习路线入口：
