## 一、MyBatis 是什么

MyBatis 是半自动 ORM 框架，开发者需要编写 SQL，框架负责映射结果。

适合场景：

- SQL 复杂
- 对查询性能与可控性要求高

---

## 二、核心概念

- `SqlSessionFactory` / `SqlSession`
- `Mapper` 接口
- XML 映射文件或注解

---

## 三、基础示例

```java
public interface UserMapper {
    @Select("select * from user where id = #{id}")
    User findById(Long id);
}
```

XML 示例：

```xml
<select id="findById" resultType="User">
  select * from user where id = #{id}
</select>
```

---

## 四、动态 SQL

```xml
<select id="query">
  select * from user
  <where>
    <if test="name != null">
      and name = #{name}
    </if>
  </where>
</select>
```

---

## 五、MyBatis-Plus

MyBatis-Plus 在 MyBatis 基础上增强，提供：

- 通用 CRUD
- 条件构造器（Wrapper）
- 分页插件

```java
List<User> users = userService.list(
    new QueryWrapper<User>().like("name", "tom")
);
```

---

## 六、学习建议

- 先掌握 MyBatis 基本用法与映射
- 再引入 MyBatis-Plus 提升效率
- SQL 复杂时优先写清晰的 XML
