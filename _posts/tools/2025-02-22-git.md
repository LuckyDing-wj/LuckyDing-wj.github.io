## 一、Git 是什么

Git 是分布式版本控制系统，适合个人与团队协作。

---

## 二、基本工作流

1. 工作区修改
2. `git add` 暂存
3. `git commit` 提交
4. `git push` 推送

---

## 三、常用命令

```
git status
git add .
git commit -m "feat: add notes"
git log --oneline
git pull
git push
```

---

## 四、分支管理

```
git branch
git checkout -b feature/login
git merge feature/login
```

建议：

- 小步提交
- 分支命名清晰（feature/bugfix）

---

## 五、解决冲突

常见步骤：

1. `git pull` 触发冲突
2. 手工解决文件冲突
3. `git add` + `git commit`
