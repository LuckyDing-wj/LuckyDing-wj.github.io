## 一、Maven 是什么

Maven 是 Java 生态最常用的构建工具，提供标准化项目结构与依赖管理能力。

---

## 二、核心概念

- `pom.xml`：项目配置入口
- 坐标（GAV）：`groupId` / `artifactId` / `version`
- 生命周期：`clean` / `default` / `site`

---

## 三、项目结构

```
src/main/java
src/main/resources
src/test/java
```

---

## 四、依赖管理

```xml
<dependencies>
  <dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
  </dependency>
</dependencies>
```

常见技巧：

- `dependencyManagement` 统一版本
- `scope` 控制依赖范围

---

## 五、常用命令

```
mvn clean
mvn test
mvn package
mvn install
```

---

## 六、插件与打包

常用插件：

- `maven-compiler-plugin`
- `maven-surefire-plugin`
- `spring-boot-maven-plugin`
