## 网站开通啦！

欢迎来到我的 Java 学习笔记网站。这是一个基于 GitHub Pages 和 Jekyll 构建的静态网站。

### 建站目的

创建这个网站的初衷是为了：

1. **系统化整理知识** - 将零散的学习笔记整理成体系
2. **分享学习心得** - 希望能帮助到同样在学习 Java 的朋友
3. **记录成长历程** - 见证自己从入门到精通的过程
