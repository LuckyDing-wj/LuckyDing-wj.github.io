## 一、阶段目标

- 理解 IoC 与 AOP 核心思想
- 掌握 Spring MVC Web 开发
- 能用 Spring Boot 搭建完整项目

---

## 二、建议学习顺序

1. Spring 核心（IoC、Bean 生命周期）
2. Spring AOP 与事务管理
3. Spring MVC 请求流程与控制器
4. Spring Boot 自动配置与项目实战

---

## 三、实践项目建议

- 实现用户增删改查 REST API
- 集成 MySQL 与 JPA / MyBatis
- 加入统一异常处理与日志

---

## 四、常见坑

- 组件扫描路径不正确
- 事务未生效（方法访问级别）
- 循环依赖
